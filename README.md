<h1 align ="center">What is Strapi?</h1>
<span style ="color:red;">What is strapi?</span>
 
   Strapi is a backend tool that lets you :
       
       1.	Create Data (like users, blogs, products
       2.	Store it in a database.
       3.	Automatically get APIs(REST/GraphQL)
       4.	Manage everything from a web admin panel
